##Como instalar el proyecto
Esta libreria requiere **Typescript** para modificar su codigo fuente. **Jasmine** es usado para correr las pruebas unitarias

##Build
Pasos para instalar:

- Instalar dependencias
>`npm install`

- Compilar Typescript
>`npm run tsc`

- Correr pruebas
>`npm test`

- Preparar para distribucion
>`npm run build`
